Paginas web

Las paginas web que creamos pueden ser estaticas o dinamicas, las paginas web estaticas son aquellas donde los usuarios no podran interactuar con la pagina y solamente podran observar el contenido de la misma. Las paginas web dinamicas son aquellas que el usuario si puede interactuar ya sea dejando comentarios o agregando mas cosas al sitio web.
